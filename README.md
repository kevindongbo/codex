# 东铂跨境：竞品监控与仓库管理

双击 `index.html` 即可使用，不需要安装软件、注册账号或连接 TikTok API。

## 最简单的使用方法

1. 点击右上角“添加商品”，填写 SKU、标准采购成本、商品链接和图片。
2. 选择“本店”时，商品会自动成为仓库 SKU；直接和间接竞品仅用于监控。
3. 在“仓库管理”中登记采购在途；确认收货后数量才会转入已在库，订单出库会扣减已在库。
4. 有公开数据时，再填写第一次快照并使用“录快照”追加竞品销量、价格与评价变化。

第一次数据只是基准。例如：

- 7 月 14 日累计销量 901
- 7 月 15 日累计销量 930
- 网页计算期间新增销量为 29

## 数据保存

数据自动保存在当前电脑的浏览器中，刷新和关闭网页后仍会保留。建议定期点击“导出数据”保存 CSV 备份。

如果更换电脑或浏览器，可以点击“导入 CSV”恢复数据。网页提供“下载 CSV 模板”，也可以用 Excel 批量填写后导入。

CSV 的 `type` 字段支持：

- `own`：本店
- `direct`：直接竞品
- `indirect`：间接竞品

## 当前支持的字段

- 商品名称、SKU、标准采购成本、店铺、商品链接、图片、市场、币种
- 在途库存、已在库、收货入库和订单出库库存流水
- 公开价格、公开累计销量
- 商品评分、评价总数、1–2 星评价数
- 店铺评分
- 自动计算期间新增销量、降价/涨价、评价变化和低星评价预警

## 注意

这里计算的是普通买家前台公开累计销量的变化，不等同于卖家后台订单数据。如果累计销量比上一次更小，网页会标记为异常，不会把它当成负销量。
